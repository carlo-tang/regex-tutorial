# Regex Tutorial

Materialien für einen praxisnahen Regex-Einstieg, gehalten als Teams-Präsentation für ein gemischtes Team aus Linux-Admins und Netzwerkkollegen ohne Kommandozeilen-Vorkenntnisse.

Plattform: [regex101.com](https://regex101.com) · Flavor: PCRE2 (PHP) · Flags: `g` `m`

---

## Dateien

**`lecture-notes.md`** — Moderationsvorlage (Source of Truth). Vollständiger Moderationstext mit allen Regexen, Erklärungen, vorbereiteten Links und Lösungen. Wird während der Vorbereitung bearbeitet.

**`aufgaben-ohne-loesungen.md`** — Kollegenversion ohne Lösungen. Wird am Ende jeder Session aus `lecture-notes.md` synchronisiert und an die Teilnehmer weitergegeben.

**`cheatsheet.md`** — Kompakte Referenzkarte zum Mitnehmen. Enthält die wichtigsten Metazeichen, Quantifizierer und Zeichenklassen auf einen Blick.

---

## Verzeichnis `kontext/`

Quellmaterial, das zur Vorbereitung gedient hat, aber nicht direkt präsentiert wird:

- **`Regular Expression Reference - Cisco.pdf`** — Cisco-Dokumentation zu PCRE (Cambridge-Doku); zu fortgeschritten für die Zielgruppe, dient als Nachschlagewerk.
- **`TLCL-25.12.pdf`** — „The Linux Command Line" als allgemeine Referenz.
- **`input-kollege.md`** — Vorbereitungsdokument eines Kollegen: erklärt Regex-Grundlagen aus Netzwerk-Perspektive (Anker, Quantifizierer, Zeichenklassen, Capture Groups) mit praktischen Netzwerk-Beispielen (IPs, MACs, Log-Einträge).
- **`cisco-regex-link.md`** — Ursprünglicher Cisco-Dokumentationslink (gibt 403 zurück; PDF liegt stattdessen im Verzeichnis).
